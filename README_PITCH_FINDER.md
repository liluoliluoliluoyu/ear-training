# 音高寻找练耳程序 (pitch_finder.py)

## 简介

这是一个音高寻找练耳程序，会持续播放音阶（循环播放），用户可以通过按键试音来寻找匹配的音高。当找到匹配音时，会听到两个音同时播放，帮助训练音高识别能力。

## 功能特点

- 🎵 **持续播放音阶**：程序会循环播放音阶，用户可以随时试音
- 🎹 **实时试音**：播放音阶的同时，用户可以通过按键试音
- 🎸 **音色区分**：音阶使用钢琴音色，用户按键使用吉他音色，便于区分
- ✅ **匹配检测**：当找到匹配音时，会听到两个音同时播放，并显示提示
- 📊 **统计信息**：显示匹配率、尝试次数等统计信息
- ⚡ **可调速度**：支持调整播放速度（0.3x - 3.0x）
- ⏸️ **暂停功能**：可以暂停/继续播放
- 🔄 **新音阶生成**：可以随时生成新的音阶

## 使用方法

### 1. 安装依赖

确保已安装必要的依赖（与 `piano.py` 相同）：

```bash
pip install pyaudio numpy pyFluidSynth
```

macOS 用户还需要安装 fluidsynth：

```bash
brew install fluid-synth
```

### 2. 运行程序

```bash
python3 pitch_finder.py
```

或使用虚拟环境：

```bash
source venv/bin/activate
python3 pitch_finder.py
```

## 按键操作

### 音符键（试音用）

- **白键**: `A` `S` `D` `F` `G` `H` `J` `K`
- **黑键**: `W` `E` `T` `Y` `U`
- **高音键**: `Q` `Z` `X` `C` `V` `B` `N` `M` `,` `.` `/` `;` `'`

### 控制键

- **空格键**: 暂停/继续播放
- **N 键**: 生成新的音阶
- **+/- 键**: 调整播放速度（+ 加快，- 减慢）
- **ESC**: 退出程序

## 使用技巧

1. **听音辨音**：先仔细听播放的音阶，记住音高
2. **试音找音**：按键盘按键试音，找到与播放音阶匹配的音高
3. **匹配提示**：当找到匹配音时，会听到两个音同时播放（钢琴音色 + 吉他音色），并显示 ✓ 提示
4. **长按试音**：可以长按按键来持续试音，更好地感受音高匹配
5. **调整速度**：如果音阶播放太快或太慢，可以使用 +/- 键调整速度

## 显示信息

程序会显示以下信息：

- **当前音阶**：显示正在播放的音阶（音符名称和频率）
- **播放速度**：当前播放速度倍数
- **按键信息**：显示用户按下的键、对应的音符和频率
- **匹配提示**：找到匹配音时显示详细信息
- **统计信息**：显示匹配率、尝试次数等

## 工作原理

1. **多线程播放**：音阶在独立线程中循环播放，不会阻塞用户输入
2. **频率匹配**：使用频率容差（1.0 Hz）来判断两个音是否匹配
3. **音色区分**：音阶使用钢琴音色，用户按键使用吉他音色，便于区分
4. **实时反馈**：找到匹配音时立即显示提示，并可以听到两个音同时播放

## 注意事项

- 确保在交互式终端中运行（不是 IDE 的内置终端）
- 如果使用 SoundFont，确保 SoundFont 文件路径正确
- 按键时注意区分音阶播放和用户按键的声音
- 可以长按按键来持续试音，但会自动在1秒后释放

## 与 ear_training.py 的区别

- **ear_training.py**：播放固定序列，用户输入完整答案进行验证
- **pitch_finder.py**：持续播放音阶，用户可以随时试音找音高，更灵活

## 故障排除

如果遇到问题，请参考 `piano.py` 的故障排除部分，因为两个程序使用相同的音频系统。
